# Spring Webflux demo

A simple spring webflux demo for study purposes.

The project is hosted on [Heroku](https://www.heroku.com/) and you can
access it [here](https://spring-webflux-demo.herokuapp.com/).

### Technologies
- Spring Boot 2.4.2
- Swagger 2.9.2
- JUnit 4.13.1
- Mockito 3.2.4
- Lombok 1.18.16
